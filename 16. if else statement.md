# Control Flow – if, elif, else Statements in Python
Control flow lets you make decisions in your code using conditions.

## Basic if Statement
x = 10
if x > 5:
    print("x is greater than 5")

## if...else Statement
x = 3
if x > 5:
    print("x is greater than 5")
else:
    print("x is 5 or less")

## if...elif...else Statement
x = 0
if x > 0:
    print("Positive")
elif x < 0:
    print("Negative")
else:
    print("Zero")
