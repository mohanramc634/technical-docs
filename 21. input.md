# Accepting Input from a User in Python
Python uses the input() function to take input from the user.

## Basic Usage

name = input("Enter your name: ")
print("Hello", name)

=> By default, input() returns data as a string.


## Convert to Integer:
age = int(input("Enter your age: "))
print("You will be", age + 1, "next year.")

## Convert to Float:
price = float(input("Enter the price: "))
print("Price + Tax:", price * 1.18)

## Example: Multiple Inputs in One Line
x, y = input("Enter two numbers: ").split()
print("Sum =", int(x) + int(y))

## Example: Taking List Input
numbers = input("Enter numbers separated by space: ").split()
numbers = [int(num) for num in numbers]
print("You entered:", numbers)
