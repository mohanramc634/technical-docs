
To call Python code from another file, you can use the concept of importing modules. Here's how:

## Example Setup
Suppose you have two files in the same directory:
my_module.py
main.py

## my_module.py
def greet(name):
    return f"Hello, {name}!"

## main.py
import my_module
message = my_module.greet("Somu")
print(message)  # Output: Hello, Somu!

## Ways to Import
1. Import Entire Module
import my_module
my_module.greet("Jeyvarun")

2. Import Specific Function
from my_module import greet
greet("Jeyvarun")

3. Import with Alias

import my_module as mm
mm.greet("Jeyvarun")

📌 Note
    • Make sure both .py files are in the same folder or the module is in the Python module search path.
If in a subfolder, use a package with __init__.py.
