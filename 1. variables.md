## Valid examples:
name = "Alice"
_age = 25
user123 = "Bob"

## Assigning Values
x = 5                    # Integer
name = "Tom"   # String
price = 12.5   # Float
is_valid = True  # Boolean

## Multiple Assignments
a, b, c = 1, 2, 3
x = y = z = 0  # Same value to multiple variables

## Checking Type
Use type() to find the type of a variable:

x = "hello"
print(type(x))   # <class 'str'>

## Changing Value
You can change a variable's value anytime:

x = 10
x = "Now I'm a string"
