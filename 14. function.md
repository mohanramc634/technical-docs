
# Functions in Python
A function is a reusable block of code that performs a specific task.

## Defining a Function
Use the def keyword:

def greet():
    print("Hello!")

## Calling a Function
greet()   # Output: Hello!

## Function with Parameters
def greet(name):
    print("Hello", name)

greet("Alice")   # Output: Hello Alice

## Function with Return Value
def add(a, b):
    return a + b
result = add(3, 5)
print(result)    # Output: 8

## Default Parameters
def greet(name="Guest"):
    print("Hello", name)

greet()            # Hello Guest
greet("Rahul")     # Hello Rahul

## Multiple Return Values

def calc(a, b):
    return a + b, a * b
sum_, product = calc(4, 5)
print(sum_, product)   # Output: 9 20

## Function with *args (Variable Arguments)
def total(*numbers):
    return sum(numbers)
print(total(1, 2, 3, 4))   # Output: 10

## Function with **kwargs (Keyword Arguments)
def info(**data):
    for key, value in data.items():
        print(key, value)
info(name="Tom", age=25)

Output: name Tom
        age 25
