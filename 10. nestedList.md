
# Accessing Elements in a Nested List (List of Lists) in Python
A nested list is a list that contains other lists as its elements.

## Example Nested List:

matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

## Accessing Elements
print(matrix[0])    # [1, 2, 3]
print(matrix[2])    # [7, 8, 9]

## Access a specific element:
print(matrix[0][0])  # 1
print(matrix[1][2])  # 6
print(matrix[2][1])  # 8

## Looping Through a Nested List
=> Print all elements row by row:
for row in matrix:
    print(row)

=> Print each element one by one:
for row in matrix:
    for element in row:
        print(element, end=" ")

=> Example (Real-world style):
grades = [
    ["Alice", 85, 90],
    ["Bob", 78, 88],
    ["Cara", 92, 95]
]
print(grades[1][0])  # 'Bob'
print(grades[2][2])  # 95
