
# Dictionaries in Python
A dictionary is a built-in data type in Python that stores key-value pairs.

## Dictionary Basics
person = {
    "name": "Alice",
    "age": 30,
    "city": "New York"
}
    • Keys must be unique
    • Keys can be strings, numbers, or tuples
    • Values can be any data type

## Accessing Values
print(person["name"])     # Alice
print(person.get("age"))  # 30
    => .get() is safer – it won’t raise an error if the key is missing.

## Changing Values
person["age"] = 31

## Adding Items
person["email"] = "alice@example.com"

## Removing Items
person.pop("city")        # Removes 'city'
del person["name"]        # Removes 'name'
person.clear()            # Empties the dictionary

## Looping Through a Dictionary
for key in person:
    print(key, person[key])
for key, value in person.items():
    print(key, value)
for value in person.values():
    print(value)

## Dictionary Length
print(len(person))   # Number of key-value pairs

## Useful Dictionary Methods
person.keys()      # Returns all keys
person.values()    # Returns all values
person.items()     # Returns all (key, value) pairs
person.update({"age": 35, "city": "Boston"})  # Update multiple items

## Example
student = {
    "name": "Rahul",
    "marks": [85, 90, 88],
    "pass": True
}
print(student["marks"][1])  # 90
