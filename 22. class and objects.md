# Classes and Objects in Python – OOP Basics
Python is an object-oriented programming (OOP) language.
It allows you to model real-world things using classes (blueprints) and objects (instances).

## What is a Class?
A class is a blueprint for creating objects. It defines attributes (variables) and methods (functions).

## What is an Object?
An object is a specific instance of a class with its own values.

## Defining a Class

class Person:
    def __init__(self, name, age):  # Constructor
        self.name = name            # Attribute
        self.age = age

    def greet(self):                # Method
        print(f"Hello, I’m {self.name} and I’m {self.age} years old.")


## Creating Objects
p1 = Person("Alice", 25)
p2 = Person("Bob", 30)
p1.greet()  # Hello, I’m Alice and I’m 25 years old.
p2.greet()  # Hello, I’m Bob and I’m 30 years old.

🔧 Key Concepts
Term	    Description
class	    Blueprint for creating objects
object	    Instance of a class
__init__	Constructor method (runs automatically)
self	    Refers to the current object (like this)
method	    A function defined inside a class
attribute	A variable attached to an object


## Example with Behavior

class Dog:
    def __init__(self, name):
        self.name = name

    def bark(self):
        print(self.name + " says woof!")

d = Dog("Bruno")
d.bark()  # Bruno says woof!
