
# Indexing and Slicing Strings in Python

## What is a String?
A string is a sequence of characters.
Example:

text = "Hello"

## Indexing
Each character in a string has an index:

 H   e   l   l   o
[0] [1] [2] [3] [4]
[-5][-4][-3][-2][-1]

### Example:
text = "Hello Mohanraj"
print(text[0])   # H
print(text[-1])  # o

### Slicing
Syntax:
string[start:stop]  # gets characters from start to stop-1

### Examples:
text = "Hello, World!"
print(text[0:5])   # Hello
print(text[7:])    # World!
print(text[:5])    # Hello
print(text[::2])   # Hlo ol!
print(text[::-1])  # !dlroW ,olleH (reversed string)

## Slice Parameters
string[start : stop : step]
    • start: where to begin (default = 0)
    • stop: where to end (not included)
    • step: how many steps to jump (default = 1)

### More Examples
text = "Python"
print(text[1:4])   # yth
print(text[::3])   # Ph
print(text[-3:])   # hon
