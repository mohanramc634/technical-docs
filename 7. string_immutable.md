
## Strings are Immutable
This means:
    Once a string is created, it cannot be changed (modified) in place.

## Example:

name = "Alice"
name[0] = "M"    # ❌ This will raise an error
📛 Error:
TypeError: 'str' object does not support item assignment

## But You Can Reassign:
You can create a new string by modifying and assigning it to the same variable:

name = "Alice"
new_name = "M" + name[1:]  # 'Mlice'

## Example (Changing a character):

text = "hello"
text = text.replace("h", "y")  # ✅ creates a new string 'yello'

Why are strings immutable?
    • Safety: Avoids bugs due to accidental changes.
    • Hashing: Makes strings usable as keys in dictionaries and sets.
Efficiency: Python can optimize memory for reusable strings.
