
# pass Statement in for Loops (and in general)
The pass statement in Python is a placeholder. It does nothing when executed and is used when a statement is syntactically required but you don’t want any action to be taken.

## Syntax:
for item in range(5):
  pass  # does nothing for each item

## Why use pass?
    • To write empty loops or functions while planning or debugging
    • To avoid syntax errors when a block is needed

## Example 1: for loop with pass
for i in range(3):
    print("Loop", i)
    pass  # placeholder for future code

## Example 2: if with pass
for i in range(5):
    if i == 3:
        pass  # You plan to handle this case later
    else:
        print(i)

## Compare with continue and break
### Keyword	Meaning
pass	    Do nothing, just a placeholder
continue	Skip the rest of the loop for this iteration
break	    Exit the loop completely
