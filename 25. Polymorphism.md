Polymorphism in Python
Polymorphism means "many forms." In Python, it allows different classes to define methods with the same name but potentially different behavior.
This enables writing flexible and reusable code.

## Example 1: Same Method Name, Different Classes

class Dog:
    def speak(self):
        return "Woof!"
class Cat:
    def speak(self):
        return "Meow!"
        
# Polymorphic behavior
for animal in (Dog(), Cat()):
    print(animal.speak())

=> Output:
Woof!
Meow!
🔹 Both classes define a speak() method, but the behavior differs.
