# List Assignment in Python

## 1. Simple Assignment
a = [1, 2, 3]
b = a
Now, a and b refer to the same list in memory.
If you modify b, a is also changed:

b[0] = 100
print(a)  # [100, 2, 3]

# 2. Copying a List (to avoid shared reference)
To create a new, independent copy:

Method 1: Using copy()
a = [1, 2, 3]
b = a.copy()
b[0] = 100
print(a)  # [1, 2, 3]

Method 2: Using slicing [:]
a = [1, 2, 3]
b = a[:]

Method 3: Using list()
b = list(a)

## 3. Multiple Assignment
a, b, c = [10, 20, 30]
print(a)  # 10
print(b)  # 20
print(c)  # 30

## 4. Unpacking with * Operator
numbers = [1, 2, 3, 4, 5]
a, *b = numbers
print(a)  # 1
print(b)  # [2, 3, 4, 5]

## 5. Assigning Values to Multiple List Elements
a = [0] * 5
print(a)  # [0, 0, 0, 0, 0]
