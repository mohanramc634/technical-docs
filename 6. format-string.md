
# Python format() Method — String Formatting
The format() method lets you insert values into a string using placeholders {}.

## Basic Usage
name = "Alice"
age = 25
print("My name is {} and I am {} years old.".format(name, age))
Output: My name is Alice and I am 25 years old.

## Using Index Numbers
print("The first value is {0}, the second is {1}".format(10, 20))
Output: The first value is 10, the second is 20

## Using Named Placeholders
print("Name: {name}, Age: {age}".format(name="Bob", age=30))
Output: Name: Bob, Age: 30

## Formatting Numbers
price = 49.56789
print("Price: {:.2f}".format(price))  # 2 decimal places
Output: Price: 49.57

## Padding and Alignment
print("{:<10}".format("Left"))    # Left aligned
print("{:^10}".format("Center"))  # Center aligned
print("{:>10}".format("Right"))   # Right aligned

## Example with All
template = "Product: {name}, Price: ${price:.2f}"
print(template.format(name="Pen", price=12.3456))
# Output: Product: Pen, Price: $12.35
