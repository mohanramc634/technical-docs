
Here are the basic arithmetic operations in Python:

## 1. Addition (+)
a = 5 + 3
print(a)  # 8

## 2. Subtraction (-)
b = 10 - 4
print(b)  # 6

## 3. Multiplication (*)
c = 7 * 2
print(c)  # 14

## 4. Division (/)
Always returns a float, even if the result is a whole number.
d = 8 / 2
print(d)  # 4.0

## 5. Floor Division (//)
Returns the whole number part (removes decimal).
e = 9 // 2
print(e)  # 4

## 6. Modulus (%)
Returns the remainder of a division.
f = 10 % 3
print(f)  # 1

## 7. Exponentiation (**)
Raises a number to a power.
g = 2 ** 3
print(g)  # 8

## Combine with Variables
x = 10
y = 3
print(x + y)   # 13
print(x - y)   # 7
print(x * y)   # 30
print(x / y)   # 3.333...
print(x // y)  # 3
print(x % y)   # 1
print(x ** y)  # 1000
