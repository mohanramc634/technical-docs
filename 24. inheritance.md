## Inheritance in Python (OOP Concept)
Inheritance allows a class (called the child or derived class) to inherit attributes and methods from another class (called the parent or base class). This promotes code reuse, organization, and helps follow DRY (Don't Repeat Yourself) principles.

## 1. Basic Inheritance Syntax
class Parent:
    def greet(self):
        print("Hello from Parent!")

class Child(Parent):  # Inheriting from Parent
    def introduce(self):
        print("I am the Child.")

c = Child()
c.greet()      # Inherited from Parent
c.introduce()  # Defined in Child

## 2. Constructor Inheritance (__init__)
If both parent and child define __init__, you must explicitly call the parent's __init__.

class Person:
    def __init__(self, name):
        self.name = name

class Student(Person):
    def __init__(self, name, roll_no):
        super().__init__(name)        # Call parent constructor
        self.roll_no = roll_no

s = Student("Alice", 101)
print(s.name)      # Alice (from Person)
print(s.roll_no)   # 101 (from Student)

## Using super()
To call the parent's method from the child class:

class Parent:
    def greet(self):
        print("Hello from Parent")
class Child(Parent):
    def greet(self):
        super().greet()
        print("Hello from Child")

c = Child()
c.greet()

=> Output:
Hello from Parent
Hello from Child

## 4. Method Overriding
Child classes can override methods from the parent class.

class Animal:
    def speak(self):
        print("Animal speaks")

class Dog(Animal):
    def speak(self):
        print("Dog barks")  # Overrides Animal's speak
d = Dog()
d.speak()  # Output: Dog barks
