
Here are the basic data types in Python:

## 1. Integer (int)
Whole numbers (positive or negative) without decimals.
age = 25
count = -10

## 2. Float (float)
Decimal numbers (floating-point).
price = 99.99
temperature = -5.5

## 3. String (str)
Text data enclosed in single (') or double (") quotes.
name = "Alice"
greeting = 'Hello, World!'

## 4. Boolean (bool)
Represents either True or False.
is_active = True
is_logged_in = False

## 5. List (list)
Ordered, changeable collection (can contain different data types).
fruits = ["apple", "banana", "cherry"]
numbers = [1, 2, 3, 4.5, "five"]

## 6. Tuple (tuple)
Ordered, unchangeable collection.
coordinates = (10, 20)

## 7. Dictionary (dict)
Key-value pairs, unordered and changeable.
person = {"name": "John", "age": 30}

## 8. Set (set)
Unordered collection of unique items.
colors = {"red", "green", "blue"}

## 9. None Type
Represents the absence of a value.
result = None

## Want to check type?
Use type():
x = 5
print(type(x))   # <class 'int'>
