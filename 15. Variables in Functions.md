
# Basics of Variables in Functions in Python
When you use variables inside functions, understanding scope is very important.

## 1. Local Variables
    • Declared inside a function
    • Only accessible within that function

def greet():
    name = "Alice"     # local variable
    print("Hello", name)
greet()                # Output: Hello Alice
# print(name)          Error: name is not defined

## 2. Global Variables
    • Declared outside all functions
    • Can be accessed inside functions, but not modified unless you use global

name = "Bob"           # global variable
def greet():
    print("Hello", name)
greet()                # Output: Hello Bob

## 3. Modifying Global Variable
To change a global variable inside a function, use the global keyword:

count = 0
def increment():
    global count
    count += 1
increment()
print(count)           # Output: 1

## 4. Function Parameters as Variables
Function parameters are local variables that receive their values when the function is called.

def greet(person):     # 'person' is a local variable
    print("Hi", person)
greet("Rahul")         # Output: Hi Rahul

## Summary
Type	Defined Where	    Scope
Local	Inside a function	Only inside that function
Global	Outside functions	Whole file/module
