
# List in Python – The Basics
A list is a built-in Python data type used to store multiple values in a single variable. Lists are:
    • Ordered
    • Changeable (mutable)
    • Allow duplicate elements
    • Can hold mixed data types (int, str, float, etc.)

## Creating a List
fruits = ["apple", "banana", "cherry"]
numbers = [1, 2, 3, 4.5]
mixed = ["hello", 42, True, 3.14]
empty = []

## Accessing List Elements (Indexing)
print(fruits[0])    # apple
print(fruits[-1])   # cherry

## Slicing a List
print(fruits[0:2])  # ['apple', 'banana']

## Modifying Lists
fruits[1] = "blueberry"
print(fruits)  # ['apple', 'blueberry', 'cherry']

## Adding Items
fruits.append("mango")         # Add to end
fruits.insert(1, "orange")     # Insert at index 1

## Removing Items
fruits.remove("banana")  # Remove by value
fruits.pop()             # Remove last item
fruits.pop(0)            # Remove by index
del fruits[1]            # Delete by index

## Looping Through a List
for fruit in fruits:
    print(fruit)

## Common List Methods
len(fruits)         # Get number of items
fruits.sort()       # Sort list
fruits.reverse()    # Reverse list
fruits.clear()      # Empty the list
fruits.copy()       # Create a copy

## List with range() and list()
nums = list(range(5))  # [0, 1, 2, 3, 4]
