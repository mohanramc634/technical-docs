
# Class atribute and object attribute
In Python, class attributes and object attributes are both used to store data, but they behave differently in terms of scope and how they're accessed.

## Class Attributes
    • Belong to the class itself, and are shared among all instances.
    • Changing a class attribute affects all instances, unless an instance overrides it.

## Example:
class Dog:
    species = "Canis familiaris"  # class attribute
    def __init__(self, name):
        self.name = name  # object attribute

dog1 = Dog("Buddy")
dog2 = Dog("Lucy")
print(dog1.species)  # Canis familiaris
print(dog2.species)  # Canis familiaris
Dog.species = "Canine"
print(dog1.species)  # Canine
print(dog2.species)  # Canine

## Object Attributes
    • Belong to the specific instance of a class.
    • Each object can have different values for its attributes.
    
Example:
dog1 = Dog("Buddy")
dog2 = Dog("Lucy")
print(dog1.name)  # Buddy
print(dog2.name)  # Lucy

🔍 Summary Table
Attribute Type	Defined Where	Shared Across Instances	Example
Class Attribute	Inside class, outside __init__	✅ Yes	species = "Canis"
Object Attribute	Inside __init__ or other methods using self	❌ No, instance-specific	self.name = "Buddy"
