
# Tuples in Python
A tuple is a built-in data type in Python used to store multiple items in a single variable, similar to a list, but with one key difference:

Tuples are immutable — they cannot be changed after creation.

## Creating a Tuple
t1 = (1, 2, 3)
t2 = ("apple", "banana", "cherry")
t3 = (1, "hello", 3.14, True)
Even a single-item tuple needs a comma:

t_single = (5,)   # ✅ tuple
not_a_tuple = (5) # ❌ just an integer

## Accessing Elements
Tuples are indexed just like lists:
t = (10, 20, 30)
print(t[0])   # 10
print(t[-1])  # 30

## Slicing Tuples
t = (1, 2, 3, 4, 5)
print(t[1:4])   # (2, 3, 4)

## Tuples are Immutable
t = (1, 2, 3)
t[0] = 10   # ❌ TypeError: 'tuple' object does not support item assignment

## Looping Through Tuples
for item in t:
    print(item)

## Tuple Methods
Tuples have only two built-in methods:

t = (1, 2, 2, 3)
print(t.count(2))   # 2
print(t.index(3))   # 3
